## Quick Start

- For USB2CANFD tools and basic connectivity, start with [dm-tools/USB2CANFD/README.md](https://github.com/dmBots/dm-tools/blob/master/USB2CANFD/README.md), then continue into `SETUP.md`.
- For host software packages and basic connectivity, open [dm-tools/USB2CANFD/SETUP.md](https://github.com/dmBots/dm-tools/blob/master/USB2CANFD/SETUP.md), then follow the setup notes.
- For `slcan` firmware flashing, switching, or validation, open [dm-tools/USB2CANFD/Firmware/socketcan/slcan-firmware/README.md](https://github.com/dmBots/dm-tools/blob/master/USB2CANFD/Firmware/socketcan/slcan-firmware/README.md), then follow `FLASHING.md`.
- For SDK materials, use [dm-tools/USB2CANFD/SDK/README.md](https://github.com/dmBots/dm-tools/blob/master/USB2CANFD/SDK/README.md) for the mirrored USB2CANFD path or [dm-tools/DM_DeviceSDK/README.md](https://github.com/dmBots/dm-device-sdk) for the broader device SDK path.
- For repository coverage questions, use [dm-tools/README.md](https://github.com/dmBots/dm-tools) together with [../docs/repository/mirror-scope.md](../docs/repository/mirror-scope.md).
- README provides navigation; `SETUP.md` provides installation and runtime preparation; `WORKFLOW.md` provides usage sequence; `FLASHING.md` provides firmware procedures.
