## 下载

1. 如果您想下载整个库，推荐使用git来进行克隆，并且将所有子仓库一起克隆

```c
git clone https://gitee.com/kit-miao/damiao.git
git submodule update
```

2. 如果您只想下载某个子仓库，可以点击你想下载的文件夹跳转到某个子仓库进行下载。

```c
git clone <url>
```
