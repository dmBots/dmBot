## Maintenance Rules

- The network `01_主资料区/` folder is the single source of truth.
- Parent repositories only maintain navigation, README files, and submodule links.
- Independent products, modules, tools, SDKs, firmware, and open-source projects are maintained as subrepositories.
- The Gitee landing README is Chinese, the GitHub landing README is English, and the other language is kept as a companion file.
