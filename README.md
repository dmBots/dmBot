# DAMIAO Public Materials

[Chinese version](README_zh.md)

This repository is the public entry point for DAMIAO materials. The network `01_主资料区/` folder is the single source of truth. Gitee uses Chinese navigation paths, GitHub uses English navigation paths, and both sides are generated from one publishing map.
