## 维护规则 / Maintenance Rules

- 网盘 `01_主资料区/` 是唯一资料源头，不在本地、Gitee、GitHub 分别维护多份资料。
- 父仓只维护分类目录、README 导航和子模块挂载关系。
- 每个独立产品、模块、工具、SDK、固件或开源项目都应独立建仓，再挂载到父仓对应目录。
- Gitee 和 GitHub 的 README 内容必须保持一致；差异只允许体现在中文路径和英文路径展示上。
- The network `01_主资料区/` folder is the single source of truth.
- Parent repositories only maintain navigation, README files, and submodule links.
- Independent products, modules, tools, SDKs, firmware, and open-source projects are maintained as subrepositories.
- Gitee and GitHub README content must stay consistent; only the displayed Chinese and English paths differ.
