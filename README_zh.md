## 维护规则

- 网盘 `01_主资料区/` 是唯一资料源头，不在本地、Gitee、GitHub 分别维护多份资料。
- 父仓只维护分类目录、README 导航和子模块挂载关系。
- 每个独立产品、模块、工具、SDK、固件或开源项目都应独立建仓，再挂载到父仓对应目录。
- Gitee 首页 README 使用中文，GitHub 首页 README 使用英文；另一种语言作为备用文件保留。
