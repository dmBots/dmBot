本页提供 `dmBot` GitHub 英文主版的中文对照入口。
如需完整中文资料，请转到 Gitee 中文主仓。

## 仓库定位

- GitHub 英文主展示仓：优先承载英文高频入口和常用公开路径。
- 不是中文完整资料主源：更完整的中文资料、历史兼容说明和更深正文仍以 Gitee `damiao` 为准。
- 目录 README 负责选入口，具体流程继续放在目录内的 `USAGE.md`、`SETUP.md`、`WORKFLOW.md`、`FLASHING.md`、`BUILD.md`、`PLATFORM.md` 中。

## 使用建议

- 英文高频入口请参见 [README.md](./README.md)。
- GitHub 与 Gitee 的覆盖差异请参见 [docs/repository/mirror-scope.md](docs/repository/mirror-scope.md)。
- GitHub 英文路径与 Gitee 中文路径对照请参见 [docs/repository/path-mapping.md](docs/repository/path-mapping.md)。
- 完整中文资料请参见 [Gitee 中文主仓](https://gitee.com/kit-miao/damiao/blob/master/README.md)。
