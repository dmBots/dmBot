# DAMIAO

[EN](./README.md) | [中文](./README.zh.md) | [Gitee 中文](https://gitee.com/kit-miao/damiao/blob/master/README.md)

这是 `dmBot` 在 GitHub 上的中文轻量入口页。  
它的职责是帮你快速判断“先看 GitHub 英文首页，还是切回 Gitee 中文主仓看完整资料”。

## 仓库定位

- GitHub 英文主展示仓：优先承载英文高频入口和常用公开路径。
- 不是中文完整资料主源：更完整的中文资料、历史兼容说明和更深正文仍以 Gitee `damiao` 为准。
- 目录 README 负责选入口，具体流程继续放在目录内的 `USAGE.md`、`SETUP.md`、`WORKFLOW.md`、`FLASHING.md`、`BUILD.md`、`PLATFORM.md` 中。

## 第一次进入请这样走

- 想直接走英文高频入口：看 [README.md](./README.md)。
- 想确认 GitHub 与 Gitee 的覆盖差异：看 [docs/repository/mirror-scope.md](docs/repository/mirror-scope.md)。
- 想把 GitHub 英文路径和 Gitee 中文路径对起来：看 [docs/repository/path-mapping.md](docs/repository/path-mapping.md)。
- 需要完整中文资料时，直接回到 [Gitee 中文主仓](https://gitee.com/kit-miao/damiao/blob/master/README.md)。
